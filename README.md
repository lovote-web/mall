## mall 

### mall 项目简介

mall 商城项目 包含 前台商城系统及后台管理系统，基于Spring Boot + MyBatis 实现。

前台商城系统包含首页，商品展示，购物车等模块。

后台管理系统包含商品管理，订单管理等模块。
